About This Project

This site was built to demonstrate core Docker containerization concepts: building a custom image, deploying a web server inside a container, and managing its lifecycle.

🐳 Containerized

Runs entirely inside an Nginx (Alpine) Docker container — no local web server installed.

⚡ Lightweight

The Alpine base image keeps the container under ~10MB of RAM at idle.

❤️ Health-Checked

A built-in HEALTHCHECK continuously verifies the server is responding correctly.
